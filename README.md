# simulator_baby

Small rest simulator service, supports ruby-2.0.0, ruby-2.1.0, ruby-2.2.0, jruby (based 1.9).

## Installation

Add this line to your application's Gemfile:

  gem 'simulator_baby'

And then execute:

  $ bundle

Or install it yourself as:

  $ gem install simulator_baby
